# DSA Name Identification Document

**Location:** App Store Connect → Digital Services Act Compliance → Name Identification Document

---

## What Documents Are Acceptable?

Apple accepts documents that **confirm your name** (Kaan Gul). You can use:

### ✅ Accepted Documents:

#### 1. **Government-Issued ID** (Best Option)
- ✅ **Turkish National ID Card** (Nüfus Cüzdanı / Kimlik Kartı)
- ✅ **Passport**
- ✅ **Driver's License**

**Requirements:**
- Must show your full name: "Kaan Gul"
- Must be current/valid (not expired)
- Must be clear and readable
- Can be in Turkish (Apple accepts multiple languages)

#### 2. **Business Registration Documents**
- ✅ **Trade Registry Certificate** (if you have a registered business)
- ✅ **Business License**
- ✅ **Chamber of Commerce Registration**

**Requirements:**
- Must show your name as owner/representative
- Must be official document
- Must be current

#### 3. **Court Documents**
- ✅ **Court Order** (if applicable)
- ✅ **Legal Name Change Document** (if applicable)

**Requirements:**
- Must be official court document
- Must show your name clearly

## Recommended: Turkish National ID Card

**Best option for Turkish citizens:**

### What to Upload:
- ✅ **Front of Turkish ID Card** (showing name, photo, ID number)
- ✅ **Back of Turkish ID Card** (if needed to show full information)

### How to Prepare:
1. **Take a clear photo** or scan of your ID card
2. **Make sure it's readable:**
   - Good lighting
   - No glare
   - Full card visible
   - Name clearly visible: "Kaan Gul"

3. **File Format:**
   - ✅ PDF (best)
   - ✅ JPEG/JPG
   - ✅ PNG
   - ❌ Maximum 10MB

4. **Privacy:**
   - You can **blur/cover** sensitive information like:
     - ID number (if you want)
     - Photo (if you want)
     - Other personal details
   - **BUT:** Name must be clearly visible
   - **BUT:** Document must look authentic

## Step-by-Step Upload Process

### Step 1: Prepare Your Document

**Option A: Turkish National ID Card**
1. Take a clear photo of your ID card
2. Make sure "Kaan Gul" is clearly visible
3. Save as PDF, JPEG, or PNG
4. Check file size (must be under 10MB)

**Option B: Passport**
1. Open passport to photo page
2. Take a clear photo or scan
3. Make sure name is clearly visible
4. Save as PDF, JPEG, or PNG
5. Check file size (must be under 10MB)

### Step 2: Upload

1. **Click** the upload area (or drag and drop)
2. **Select** your document file
3. **Wait** for upload to complete
4. **Verify** the document is visible in the preview
5. **Click "Next"** to continue

### Step 3: Document Language

- **Select:** "English (US)" or "Turkish"
- Apple accepts both, but English might be faster to verify
- If your document is in Turkish, you can still select "English (US)" - Apple has translators

## Common Issues & Solutions

### Issue: "Document not clear"
**Solution:**
- Take a new photo with better lighting
- Use a scanner instead of phone camera
- Make sure document is flat (no wrinkles)

### Issue: "Name doesn't match"
**Solution:**
- Make sure the name on document exactly matches "Kaan Gul"
- If you have a middle name on document, that's usually fine
- If name is different, you may need to update your Apple account name first

### Issue: "Document expired"
**Solution:**
- Use a current/valid document
- Renew your ID card if expired
- Use passport if ID is expired

### Issue: "File too large"
**Solution:**
- Compress the image
- Use PDF instead of high-res JPEG
- Reduce image quality slightly (but keep it readable)
