# Moving or Renaming the Project Folder

## ‚úÖ Safe to Move/Rename

**Yes, you can safely move the project folder to another location and rename it without breaking the project.**

## üîç What I Checked

### ‚úÖ No Hard-Coded Paths in Code
- All code uses relative paths
- No absolute paths in source files
- Build configurations use relative paths

### ‚úÖ Safe Files
- **Git repository** - Uses relative paths
- **Android build files** - Use relative paths
- **iOS Xcode project** - Xcode handles path updates automatically
- **Source code** - All relative paths

### ‚ö†Ô∏è Files That Will Auto-Update

#### Android
- **`android/local.properties`** - Auto-generated by Android Studio
  - Contains SDK path (machine-specific)
  - Already in `.gitignore`
  - Will be recreated automatically when you open project

#### iOS  
- **Xcode project files** - Xcode updates paths automatically
  - `.xcodeproj` files contain paths, but Xcode manages them
  - Just open the project in new location

## üìã Steps After Moving

### 1. Android Project
```bash
# Navigate to new location
cd /new/path/to/project/android

# Android Studio will:
# - Auto-detect new location
# - Regenerate local.properties
# - Update any internal paths
```

**What to do:**
- Open Android Studio
- Open the project from new location
- Android Studio will handle everything automatically

### 2. iOS Project
```bash
# Navigate to new location
cd /new/path/to/project/ios

# Open in Xcode
open Ciphio/Ciphio.xcodeproj
```

**What to do:**
- Open Xcode
- Open the project from new location
- Xcode will update internal paths automatically

### 3. Git (if using)
```bash
# Git doesn't care about folder location
# Just continue using git normally
cd /new/path/to/project
git status  # Should work fine
```

### IDE Settings (Optional)
- **Android Studio/IntelliJ**: 
  - If you see "VCS root configuration problems" warning:
    - Go to Settings ‚Üí Version Control ‚Üí Directory Mappings
    - Remove the mapping for `CiphioVault` (it's not a separate Git repo)
    - Keep only the parent directory (`/Users/qaantum/Desktop/Project`) mapped to Git
    - The IDE will auto-detect the Git repo from the parent directory
- **Xcode**: May need to re-index (automatic)
- **VS Code/Cursor**: Should auto-detect new location
