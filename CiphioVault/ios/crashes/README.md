# iOS Crash Reports

This folder contains iOS crash reports and crash logs for debugging purposes.

## How to Add Crash Reports

1. **From Xcode:**
   - Window → Devices and Simulators
   - Select your device
   - View Device Logs
   - Export crash logs here

2. **From Device:**
   - Settings → Privacy & Security → Analytics & Improvements → Analytics Data
   - Find your app's crash logs
   - Share/export them here

3. **From TestFlight:**
   - App Store Connect → TestFlight → Crashes
   - Download crash reports and add them here

## File Naming Convention

Use descriptive names like:
- `crash-2025-11-30-autofill-extension.txt`
- `crash-2025-11-30-password-decryption.crash`
- `crash-2025-11-30-keychain-access.txt`
